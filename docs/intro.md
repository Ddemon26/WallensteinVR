# Introduction

WallensteinVR is a small VR demo written in Zig. It relies on Vulkan for graphics
and OpenXR for device support.

## Features

- Basic OpenXR context creation
- Vulkan setup with debug layers
- Simple event loop (work in progress)
- Modular `xr` and `vulkan` components
- Physical device selection
- Cross-platform surface support
- Logical device initialization with queue access
- Expanded XR helpers and cleaner Vulkan context

Use the Usage Guide for build instructions.
